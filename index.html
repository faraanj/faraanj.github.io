What projects have I completed

Here is a compilation of the software projects I have successfully completed. These projects demonstrate my proficiency in programming and highlight my ability to deliver high-quality solutions. From web applications to software tools to games, each project reflects my dedication to producing reliable and efficient products.

Image Filter

This is a C-based webpage that allows for you to add a filter such as greyscale, gaussian blur, or edge detection to an uploaded bitmap file.

Accessible Boggle Game

This is a Boggle game application in Java with a focus on user accessibility for individuals with visual impairments accomplisded with additional features such as the ability to resize components of the application.

Sokoban Lite Game

This is a puzzle game written entirely in Assembly mimicking the core gameplay of the Japanese puzzle game Sokoban. Players try to find a way to push a box onto its target while avoiding corners and walls.

DavidXGoliath Game

This is a first-person shooter game application in Python using the Pygame library. Players must take down a never ending onslaught of enemies while avoiding being trapped against the game walls.

Snake Blockade Game

This is a Snake game application made in Java using the Swing graphical user interface toolkit. The game is desinged to help educate computer science students on various topics in a fun and interactive way.

Investment Calculator

This is a calculator application made in Java using the Swing graphical user interface to help those who want to see how much money would be saved when their money is compunded over a variety of time periods.
